# 👀 Staring Eyes

A fun CSS-only interactive eye effect that follows your cursor movement!

## 🔗 Live Demo
[👉 Click here to see it live](https://sabbir-ahmed-omi.github.io/Staring-Eyes/)

## ✨ Features
- Eyes follow your mouse cursor
- Smooth blinking animation
- Built with pure HTML & CSS (no JavaScript!)

## 🛠️ Tech Used
- HTML
- CSS

## 📂 How to Run Locally
Just open `index.html` in your browser!
